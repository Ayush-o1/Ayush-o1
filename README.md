### [⚡ ContextForge](https://github.com/Ayush-o1/contextforge)

**OpenAI-compatible LLM proxy middleware that cuts API costs by up to 60% — with zero code changes.**

Point your app at `localhost:8000` instead of `api.openai.com`. Same SDK, same API, same code. Behind the scenes:

```
Your App → ContextForge Gateway
              ├─ Context Compression   (summarize long conversations → fewer tokens)
              ├─ Semantic Cache         (FAISS + Redis, ≥92% cosine similarity → <30ms)
              ├─ Smart Model Router     (simple → gpt-3.5 | complex → gpt-4o)
              ├─ Telemetry              (SQLite, every request logged locally)
              └─ Response ← upstream API
```

Ships with a real-time analytics dashboard. 45+ commits. Fully containerized.
