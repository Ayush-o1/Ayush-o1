### Selected Work

&nbsp;

**[ContextForge](https://github.com/Ayush-o1/contextforge)** `FastAPI · Docker · Redis · Railway`

OpenAI-compatible LLM proxy gateway with semantic caching, multi-provider routing, and configurable fallback/retry logic. Context compression reduces token overhead before requests reach the model. Deployed continuously to Railway via GitHub Actions CI/CD. Designed to be a drop-in layer between any application and any LLM provider.

&nbsp;

**[FreightFlow](https://github.com/Ayush-o1/FreightFlow)** `Node.js · Express · MongoDB · JWT`

Production-style logistics REST API with role-based access control across shipper, driver, and admin actor types. JWT authentication with structured middleware, clean route/controller/model separation, and defined permission boundaries at every endpoint. Built to demonstrate real backend architecture, not just CRUD.

&nbsp;

**[AutoPrompt](https://github.com/Ayush-o1/autoprompt)** `FastAPI · Docker · GitHub Actions · Gemini`

Prompt optimization and evaluation engine with YAML-driven template versioning and automated scoring pipelines. Treats prompt engineering as a software engineering problem — reproducible, versioned, testable. CI/CD pipeline runs evaluations on every change. Integrates Gemini API.

&nbsp;

**[Onboarder](https://github.com/Ayush-o1/Onboarder)** `Node.js · React · Docker`

Paste a GitHub URL, receive a working Docker environment in minutes. Eliminates the entire manual setup phase for new contributors joining MERN stack projects. Turns "clone and follow 15 steps" into a single command.
